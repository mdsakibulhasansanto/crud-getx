Flutter CRUD Application

A simple Flutter-based CRUD (Create, Read, Update, Delete)

---

 🚀 Features

- 📦 Create product
- 📄 Read product list
- ✏️ Update product
- ❌ Delete product
- 🔁 Refresh list using `RefreshIndicator`
- 📡 Integrated with live REST API
- 🧠 State management using GetX
- 🎯 Clean UI with Material Design

---

🛠️ Tech Stack

- Flutter
- Dart
- GetX  for state management
- REST API  (e.g., [crud.teamrabbil.com](https://crud.teamrabbil.com/))
- HTTP package for networking
